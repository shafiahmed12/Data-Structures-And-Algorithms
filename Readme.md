# Data Structures and Algorithms

## What is a data structure

A data structure is a collection of values. The values can have relationships among them or they can have functions applied to them. Each Data structure is different in terms of what it can do and what it can be used for. Most important thing to remember is that each data structure is good and is best at what it does.

You can think of data structure as a compartment or a container. For example you have a bag pack where you put all your school related stuff, A drawer where you put your clothes, or a fridge where all your drinks and other veggies are stored or just a folder for your files or a siple box to store something.

Each of the above is meat to do a specific thing in the same way there are different data structure in which you can organize data and find them when needed.

There are 2 ways to understand Data structures.

1. How to build one and
2. How to use it.
